# froede-protocol

The wire protocol shared by the [froede](https://github.com/Mun1to/froede) browser extension and its local companion: zod schemas for every message (`ping`, `write-text`, `write-style`, `write-attr`), including the style-property and attribute allowlists that keep source-file splicing injection-safe.

You only need this package if you are building on top of froede itself. To *use* froede, see [github.com/Mun1to/froede](https://github.com/Mun1to/froede#readme).
